# CyberShield Documentation

## Project Overview

CyberShield is a cyber-integrated safety platform for women, developed for KANADSHIELD26_P2_01.

## Components

- **mobile/** — Flutter app (Android/iOS)
- **backend/** — FastAPI REST + WebSocket API
- **dashboard/** — React police command center
- **ai-engine/** — ML microservices

## API Documentation

Once running, visit http://localhost:8000/docs for interactive Swagger docs.
