Police Dashboard (React) — CyberShield

Suggested stack: React + Vite, Leaflet (map), Recharts (analytics), Tailwind CSS.

Quick start:
1. Create project: npm create vite@latest police-dashboard --template react
2. Install map & chart libs: npm i leaflet react-leaflet recharts tailwindcss
3. Implement pages: Command Center, SOS Monitor, Complaint Management, Evidence Review
4. Connect to backend APIs and WebSocket /ws/sos for real-time alerts

UI notes: Dark sidebar, white content area; KPI cards, live SOS map, recent activity feed.
