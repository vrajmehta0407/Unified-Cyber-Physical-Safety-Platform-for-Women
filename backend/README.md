Backend (FastAPI) — CyberShield

Quick start:
1. Create a virtualenv: python -m venv .venv && .\.venv\Scripts\activate
2. Install dependencies: pip install -r requirements.txt
3. Configure environment variables: copy .env.example to .env and edit
4. Start server: uvicorn app.main:app --reload --port 8000

Notes:
- This is a scaffold with placeholder endpoints. Implement DB models, auth, and integrations per spec.
